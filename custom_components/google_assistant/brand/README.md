# Brand Assets

This directory contains brand assets for the Google Assistant Unleashed integration.

## Required Files

- **icon.png** - Main icon for the integration (required by HACS)
  - Size: 256x256px or 512x512px recommended
  - Format: PNG with transparency
  - Design: Should represent Google Assistant with an "unleashed" theme

## Optional Files

- **logo.png** - Larger logo for use in documentation
- **icon@2x.png** - High-resolution icon for retina displays

## Design Guidelines

The icon should:
- Be recognizable as Google Assistant-related
- Include visual elements suggesting "unleashed" or freedom
- Work well at small sizes (as small as 48x48px)
- Use transparent background
- Follow basic Google design aesthetics

## Placeholder

Until a custom icon is created, you can use the official Google Assistant icon or any Google-themed icon that represents the integration.
